# Brute Force Attack Report

## Target
DVWA Login Page

## Attack Type
Brute Force Authentication Attack

## Tool Used
Hydra

## Findings
- Weak credentials discovered
- Username: admin
- Password: password

## Risk Level
High

## Impact
An attacker can gain unauthorized access to the system.

## Recommendation
- Enforce strong password policy
- Limit login attempts
- Implement CAPTCHA
- Enable MFA
